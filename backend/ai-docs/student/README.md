Drop internal student-facing documents here for PMS AI.

Supported file types:
- `.md`
- `.txt`
- `.text`
- `.html`

Examples:
- onboarding guide
- project workflow notes
- grading policy summary
- sprint rules
- FAQ for students

Notes:
- PMS AI will try to retrieve the most relevant excerpts from this folder per question.
- Keep documents clean and structured with short sections and headings for better answers.
- You can also keep `pj.text` in the project root; it is already indexed automatically.
